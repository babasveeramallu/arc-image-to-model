## üîß If Something Goes Wrong

**"Repository not found"**
- Make sure you pushed to GitHub
- Check the URL is correct

**"Python not found"**
- Replit handles this automatically
- Just click Run and wait

**"Camera not working"**
- Must use HTTPS (Replit does this)
- Allow camera permissions on phone
- Works on Chrome/Safari on mobile

**"Models not loading"**
- Check files are in the repo
- Make sure `models/` folder exists
- Try refreshing page
